# Student Archetypes

When testing Hero Academy balance, NEVER assume all students act the same. Simulations must inject variety using the following archetypes:

## 1. The Tryhard (Top 10%)
- **Behavior**: Completes 100% of quests. Never misses a streak. Answers correctly >90% of the time.
- **Spending**: Hoards gold, buys massive XP boosts, and strictly optimizes artifact combos.
- **Risk to Balance**: Can easily run away with the leaderboard, making others feel Hopeless. Balance to ensure their XP has diminishing returns.

## 2. The Casual (Average 50%)
- **Behavior**: Completes 70-80% of quests. Drops streaks occasionally. Makes mistakes regularly.
- **Spending**: Buys HP potions when needed, uses artifacts when reminded.
- **Risk to Balance**: Might die to the Season Boss if HP penalty is too high. Require safety nets for them.

## 3. The Slacker (Bottom 20%)
- **Behavior**: Missing 40%+ of assignments. Frequent 0 HP deaths.
- **Spending**: Often broke, cannot afford revives.
- **Risk to Balance**: High risk of permanent churn. Ensure **Comeback Potential** exists (e.g. passive XP boosts for being behind, cheap resurrects).

## 4. The Whale / Item Hoarder
- **Behavior**: Uses every single consumable, buys all lootboxes, trades aggressively if implemented.
- **Risk to Balance**: Can bypass mechanics via brute force. Limit their dominance by restricting OP artifact drop rates.
