# Game Mechanics: Система Дропа Артефактов и Балансировка

## 1. Дроп Артефактов за Задания

### 1.1 Когда срабатывает дроп?

`rollArtifactDrop` вызывается каждый раз при начислении XP учителем (`grant_xp`), т.е.:
- При выставлении оценки в Гримуаре (ДЗ или Контрольная)
- При ручном бафф через Радар Класса

### 1.2 Шанс выпадения (Шаг 1: выпадет ли что-нибудь?)

```
base = 8% + (difficulty − 1) × 2%
→ difficulty 1 = 8%, difficulty 5 = 16%

typeMult:
  quest (ДЗ)            × 0.5   → итого ~ 4–8%
  dungeon (контрольная) × 1.0   → итого ~ 8–16%
  boss (босс)           × 2.0   → итого ~ 16–32%

levelBonus = 1 + min(heroLevel × 1%, 50%)
→ Lv15 = +15%, Lv50+ = +50% (максимум)

finalChance = base × typeMult × levelBonus
```

> `drop_rate_multiplier` из раздела Балансы множит `baseDropChance` на свой %.

### 1.3 Выбор редкости (Шаг 2: какой уровень?)

Если дроп выпал — кидаются кубики по весам сложности:

| Difficulty | Common | Rare  | Epic  | Legendary |
|-----------|--------|-------|-------|-----------|
| 1         | 85%    | 14%   | 1%    | 0%        |
| 2         | 70%    | 25%   | 4.5%  | 0.5%      |
| 3         | 55%    | 33%   | 10%   | 2%        |
| 4         | 40%    | 35%   | 19%   | 6%        |
| 5         | 25%    | 35%   | 28%   | 12%       |

### 1.4 Выбор артефакта (Шаг 3: какой конкретно?)

Из каталога берутся артефакты выбранной редкости с `min_level <= heroLevel`. Один выбирается случайно с равным весом. Поле `drop_rate` в таблице `artifacts` **в этом шаге не используется** (используется только в лутбоксах).

Исключены: `effect = 'lootbox'`, артефакты серии `royal_set`.

## 2. Лутбоксы (Сундуки)

### 2.1 Источники лутбоксов

| Источник | Условие | Редкость сундука |
|---|---|---|
| 🔥 Стрик | 3 дня | Common |
| 🔥 Стрик | 7 дней | Rare |
| 🔥 Стрик | 14 дней | Epic |
| 🔥 Стрик | 30 дней | Legendary |
| 🐉 Убийство босса | На основе урона + уровня | Common–Legendary |
| 🛒 Магазин | Покупка за Gold | Common / Rare / Epic / Legendary |

### 2.2 Лутбокс за убийство сезонного босса

Качество сундука определяется двумя факторами:

**База по уровню героя:**

| Уровень героя | Базовый тир |
|---|---|
| Lv 1–15 | 📦 Common |
| Lv 16–30 | 🎁 Rare |
| Lv 31–45 | 💎 Epic |
| Lv 46–100 | 👑 Legendary |

**Корректировки тира по урону:**

| Условие | Изменение тира |
|---|---|
| 0% урона | −1 тир (consolation) |
| < 25% урона | базовый тир |
| ≥ 25% урона | +1 тир |
| ≥ 40% урона | +2 тира |
| MVP (максимум урона в классе) | дополнительно +1 тир |

Финальный тир = min(legendary, база + бонусы).

**Пример:** Герой Lv18 (база = Rare), MVP с 35% урона:
`Rare index=1 → +1 (≥25%) → +1 (MVP) = index=3 → Legendary`

### 2.3 Открытие лутбокса

При открытии выбирается тир редкости по весам:

| Сундук | Common | Rare | Epic | Legendary |
|---|---|---|---|---|
| 📦 Common | 80% | 20% | — | — |
| 🎁 Rare | 40% | 50% | 10% | — |
| 💎 Epic | — | 50% | 40% | 10% |
| 👑 Legendary | — | 20% | 50% | 30% |

Затем из выбранной редкости случайно (взвешенно по `drop_rate`) выбирается артефакт.

**Ограничение уровня:** при открытии лутбокса фильтруются только артефакты с `min_level <= heroLevel + 5`.
Герой Lv10 не получит артефакт с min_level 60.

## 3. Балансовые Множители (Economy Config)

| Параметр | На что влияет | Диапазон |
|---|---|---|
| `xp_multiplier` | Все XP от учителя (радар, гримуар) | 25–500% |
| `dmg_multiplier` | HP урон от ошибок | 10–300% |
| `gold_multiplier` | Gold за задания | 25–500% |
| `drop_rate_multiplier` | Базовый шанс дропа артефакта | 10–500% |
| `boss_hp_multiplier` | Максимальное HP сезонного босса | 25–400% |

Конфиг применяется каскадно: **class → school → global**. Если для класса нет своей настройки — берётся school, затем global.

---

## 4. Сезонный Сброс

В конце сезона (учебной четверти):
- Обнуляются HP героев
- Обнуляется стрик
- Сезонный босс помечается как `is_defeated = false` для нового сезона
- Топ лидерборд фиксируется
