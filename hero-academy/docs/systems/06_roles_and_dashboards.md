# 06. Архив: Игровые Роли и Их Права

В приложении существует строгая маршрутизация пользователей на основе роли (`/auth/login`).

## 1. System Admin (`/admin/...`)
* **Права:** Абсолютный контроль над инфраструктурой. Управление Школами и Классами.
* **Экономика:** Настройка каскадных множителей дропа, золота и XP через вкладку `/admin/economy`.
* **Доступ к:** Глобальному магазину (`/shop`), Логам аудита, Новостным лентам (`/news`) и Подпискам.

## 2. Teacher (Учитель / `/teacher/...`)
* **Радар Класса (`/live`)**: Дашборд для выдачи опыта и золота прямо во время урока в реальном времени. Раздает XP (и дроп) выбранным ученикам.
* **Управление Боссами (`/boss`)**: Учитель может объявить классного босса (например, на время контрольной), задать ему HP и отслеживать прогресс группы.
* **Управление Квестами (`/quests`)**: Выдача домашних заданий и тестирование.

## 3. Parent (Родитель / `/parent/...`)
* **Права:** Режим "Только чтение" (Observer Mode).
* **Дашборд:** Родитель видит историю оценок (полученных квестов), урон по здоровью (почему у ребенка 50 HP), и не может вмешиваться в игровой прогресс (запрет на отправку артефактов).

## 4. Student (Ученик / `/student/...`)
* **Инвентарь:** Управление артефактами и открытие Сундуков (`LootBoxModal`).
* **Магазин:** Трата золота на зелья и косметику.
* **Leaderboards (Гильдии):** Студенты соревнуются по XP внутри своего Класса. Топ учеников отображается на главном экране.
