# Привет! Мы остановились вот здесь:

У вас завис терминал, и мы решили массово обновить пол всем существующим ученикам прямо через панель управления базой данных, чтобы их аватарки корректно эволюционировали.

### Что нужно сделать прямо сейчас:

1. Откройте в браузере вашу панель управления **Supabase**.
2. Перейдите в раздел **SQL Editor** (иконка терминала слева в меню).
3. Нажмите "New Query" (Новый запрос).
4. Скопируйте и вставьте туда следующий SQL-код:

```sql
UPDATE heroes
SET gender = CASE
  -- Точные совпадения популярных женских имен
  WHEN lower(split_part(name, ' ', 1)) IN ('аня', 'мила', 'соня', 'даша', 'маша', 'настя', 'катюша', 'катя', 'оля', 'света', 'любовь', 'асель', 'айгуль') THEN 'female'
  -- Исключения мужских имен, оканчивающихся на 'а' или 'я'
  WHEN lower(split_part(name, ' ', 1)) IN ('илья', 'никита', 'данила', 'саша', 'леша', 'лёша', 'паша', 'миша', 'гриша', 'коля', 'вася', 'женя', 'петя', 'вова', 'тёма', 'рома', 'витя', 'дима', 'ваня', 'гоша', 'илюша', 'андрюша', 'сережа') THEN 'male'
  -- Общее правило: если имя заканчивается на 'а' или 'я' -> женский
  WHEN right(lower(split_part(name, ' ', 1)), 1) IN ('а', 'я') THEN 'female'
  -- Все остальные случаи -> мужской
  ELSE 'male'
END;
```

5. Нажмите кнопку **RUN** (Выполнить).

Все существующие герои автоматически получат правильный пол!
