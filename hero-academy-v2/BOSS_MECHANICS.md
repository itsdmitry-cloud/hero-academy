# Hero Academy — Механики Boss Battle

## 1. Что такое Сезонный Босс

Каждый сезон (четверть) для каждого класса автоматически создаётся **один Сезонный Босс на каждый предмет, который ведёт учитель данного класса**.

- Если учитель ведёт **Алгебру и Геометрию** → создаётся 2 босса.
- Боссы **не создаются вручную** учителем — они появляются автоматически при первом открытии дашборда учителя.
- В конце сезона боссы сохраняются в истории. Новый сезон = новые боссы с полными HP.

---

## 2. Расчёт HP Босса

```
HP = кол-во учеников × 500
Минимум: 3 000 HP
Максимум: 30 000 HP
```

**Пример:** В классе 12 учеников → 12 × 500 = **6 000 HP**

---

## 3. Источники урона по Боссу

Урон засчитывается только если `subject` (предмет) элемента активности совпадает с `subject_id` босса.

### 3.1 Гримуар — задания учителя

| Тип задания | Базовый урон |
|---|---|
| Базовое (ДЗ) | XP за задание × % оценки |
| Сложное | XP за задание × % оценки |
| Контрольная | XP за задание × % оценки |
| Проверочная | XP за задание × % оценки |
| Диктант | XP за задание × % оценки |

**Урон = XP начисленный ученику за эту работу** (с учётом артефактов и балансного множителя).

Таблица оценок:

| Оценка | % XP | % Gold | % HP-урон герою |
|---|---|---|---|
| 5 — Отлично | 100% | 100% | 0% |
| 4 — Хорошо | 80% | 80% | 0% |
| 3 — Средне | 50% | 50% | 20% |
| 2 — Плохо | 20% | 0% | 60% |
| 1 — Двойка | 0% | 0% | 100% |

> Урон босу = ∑(XP начисленных всем ученикам) за данное задание.

### 3.2 Радар класса — активность на уроке

Учитель нажимает кнопки в режиме реального времени:

| Действие | XP герою | Урон боссу |
|---|---|---|
| ⚡ Ответил правильно | 30 XP | 30 урона |
| 🌟 Блестящий ответ | 50 XP | 50 урона |
| 📚 Работа у доски | 40 XP | 40 урона |
| ❤️ Восстановить HP | 0 XP | 0 урона |
| 💀 Штраф HP | 0 XP | 0 урона |

> Урон за радар = XP выданный герою. Артефакты XP_BOOST усиливают урон.

## 4. Модиfикаторы урона (артефакты)

| Артефакт | Эффект |
|---|---|
| XP Boost (+20%) | Герой наносит на 20% больше урона боссу |
| XP Boost (+50%) | Герой наносит на 50% больше урона боссу |

**Пример:** Ученик с артефактом XP+20% получает оценку 5 за контрольную с базовым XP=350:
- Базовый XP: 350
- Арতефакт: ×1.20 = 420
- Случайный бонус (±10%): ~420 XP
- **Урон боссу: 420**

## 5. Логика нанесения урона (API)

```
POST /api/game/action
{
  action: "grant_xp",
  hero_id: "...",
  base_amount: 350,
  subject: "Алгебра",    ← ключевое поле
  teacher_id: "..."
}
```

Алгоритм внутри API:
```
1. Найти героя → получить его class_id и school_id
2. Найти активный сезон для школы
3. Найти босса: subject_bosses WHERE class_id = X AND subject_id ILIKE subject AND season_id = Y
4. Уменьшить current_hp на finalAmount (XP после всех модификаторов)
5. Записать в boss_damage_logs
6. Если current_hp = 0 → is_defeated = true → раздать наградной пул (25 000 XP, 5 000 Gold)
```

---

## 6. Победа над Боссом

При снижении HP до 0:
- Победитель занесён в `boss_damage_logs`
- **MVP** (максимальный урон) = +2 000 XP бонус
- **Last Hit** (последний удар) = +1 000 XP бонус
- Пул наград: 25 000 XP + 5 000 Gold, делится **пропорционально урону** между всеми участниками

---

## 8. Сезон и сброс

- Администратор управляет сезонами через админ-панель
- При смене сезона (status → ended): боссы остаются с историей, новый сезон создаёт новых боссов
- HP боссов **не сбрасывается** в рамках одного сезона

---

## 9. Важно: совпадение предметов

Урон засчитывается **ТОЛЬКО** если предмет задания совпадает с предметом босса (без учёта регистра).

✅ Задание `subject: "Алгебра"` → Босс `subject_id: "алгебра"` — совпадает  
❌ Задание `subject: "Математика"` + Босс `subject_id: "Алгебра"` — не совпадает

**Решение:** при проверке задания (check page) система автоматически создаёт босса для предмета задания если его нет, через `/api/bosses/ensure`.
